> A sleek, macOS-inspired productivity dashboard for saving, categorizing, and managing your best AI prompts for tools like ChatGPT and Midjourney.

PromptHub runs entirely in your browser. By leveraging the browser's native `localStorage`, you can securely save your workflow prompts without needing a backend server, database, or login.

## ✨ Features

* **💾 Zero-Database Persistence:** Saves all prompts directly to your browser's Local Storage. Close the tab, restart your computer—your prompts will still be there.
* **📋 1-Click Copy:** A built-in copy mechanism lets you instantly grab a prompt and paste it into your favorite AI tool.
* **🔍 Instant Search & Filtering:** Filter your library by categories (ChatGPT, Midjourney, Coding, Writing) or search by keyword in real-time.
* **💎 macOS Glassmorphism UI:** Features a premium dark-mode aesthetic with frosted glass sidebars, animated meshes, and floating modals.
* **⚡ Lightweight & Fast:** Built with pure Vanilla HTML, CSS, and JavaScript. No heavy frameworks or dependencies required.

---

## 🛠️ Tech Stack

* **Structure:** Semantic HTML5
* **Styling:** CSS3 (Grid, Flexbox, Backdrop-filter for glass effects, CSS Variables)
* **Logic:** Vanilla JavaScript (ES6+)
* **Data Storage:** Web Storage API (`localStorage`)
* **Icons:** [Lucide Icons](https://lucide.dev/)
* **Typography:** [Inter](https://fonts.google.com/specimen/Inter) (Google Fonts)
